# Playwright Auth 🚀


Simple login automation project built with Python, Playwright, and Pytest.

## Tech Stack

- Python
- Playwright
- Pytest

## Features

- Positive Login Test
- Page Object Model (POM)
- Pytest Fixtures

## Project Structure

```text
playwright-auth/
├── pages/
├── tests/
├── conftest.py
├── requirements.txt
└── README.md
```

## Run

```bash
python -m pytest -v
```
